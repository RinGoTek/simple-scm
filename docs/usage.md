# 用法

&nbsp;

## 运行

在simple-scm的目录下，运行：

```
./simple-scm
```

查看用法：

```
./simple-scm --usage
```

初始化存储库：

```
./simple-scm init
```

从当前节点创建新分支：

```
./simple-scm new-branch <BranchName>
```

显示分支列表：

```
./simple-scm list
```

忽略路径

```
./simple-scm ignore <path>
```

取消忽略路径

```
./simple-scm ignore -d <path>
```
